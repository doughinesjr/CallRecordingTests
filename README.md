## Initial Setup and First Time Run


1. Install Java 8+

2. Download and install Google Chrome if Chrome is not already installed
   <https://www.google.com/chrome/>

3. Git clone this project

4. Install Packages
    `npm install`

5. Run Tests
    `npm test`
